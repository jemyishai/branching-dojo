Welcome to Bridge Shape Dojo!

Hi! This is a tool to help learners practice nested for-loops in C++. It is designed with beginning coders in mind.

Select a pattern, write C++ code to produce it, then click Run. Use the Hint panel for loop-structure guidance, and Explain ✦ to see a plain-English summary of your code. When your output doesn't match, you'll get a specific hint about why.
